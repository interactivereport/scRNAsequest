Chapter 6 Output

6.1 QuickOmics visualization

The output results can be piped to QuickOmics (http://quickomics.bxgenomics.com) for visualization. Here are the four necessary files required by QuickOmics:

SRP199678_estCount.rds

SRP199678_network.RData

SRP199678.RData

SRP199678.csv

This example has been uploaded to the QuickOmics site:

https://quickomics.bxgenomics.com/?serverfile=SRP199678

6.2 Other output files

Also, the EA pipeline generates a couple of files under the working directory.

The pdf files and index.html can be directly visualized by the user. However, we suggest the user to download the ready-to-transfer ProjectName_BookdownReport.tar.gz tarball and open the index.html file inside for visualization.

Here is an example of the output directory hierarchy:

SRP199678/

    ├── counts.tsv
